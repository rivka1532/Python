# Python Programming Course

This repository contains Python homework assignments and exercises organized by lessons. A collection of Python learning materials and practice projects.

## 📚 Course Structure

The repository is organized into lesson folders containing various Python exercises and assignments:

### Lesson 1
- **File**: `main.py`
- Basic Python programming concepts

- Introduction to Python development environment

### Lesson 3
- Advanced Python concepts
- Data structures and algorithms
- Problem-solving exercises

### Lesson 4
- Object-oriented programming

### Lessons 5-6
- Combined lesson covering:
- Advanced data structures

- Working with libraries and packages

## 🚀 Getting Started

### Prerequisites
- Python 3.x installed on your system
- Code editor (VS Code, PyCharm, or similar)
- Basic understanding of programming concepts

### Running the Code

1. Clone the repository:
   ```bash
   git clone https://github.com/rivka1532/Python.git
   cd Python
   ```

2. Navigate to any lesson folder:
   ```bash
   cd lesson1
   ```

3. Run the Python files:
   ```bash
   python main.py
   ```

## 🎯 Learning Objectives

Through these lessons, you'll learn:

- **Basic Python Syntax**: Variables, data types, operators
- **Control Structures**: If statements, loops, functions
- **Data Structures**: Lists, dictionaries, tuples, sets
- **Object-Oriented Programming**: Classes, objects, inheritance
- **File Operations**: Reading and writing files
- **Error Handling**: Try-catch blocks and exceptions
- **Python Libraries**: Working with built-in and external libraries
